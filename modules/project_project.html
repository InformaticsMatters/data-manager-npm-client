Const createProject

  • createProject(projectPostBodyBody: ProjectPostBodyBody, options?: AxiosRequestConfig<any>): Promise<ProjectPostResponse>
    • Defined in project/project.ts:102

    Creates a new project, assigning and returning a unique ID. A peristent volume is created in the cluster and assigned to your Project. It's here you can add Datasets as Files and run Applications and Jobs.

    To create a Project you will need an uncommitted Project product.

    As the project owner (creator) you are also automatically an editor of the project and can add other users as editors so that they can also manipulate the project. An editor of a project can also delete it.

    The name you give the project is free-form text (can contain spaces etc.). The name you use must be unique amongst all the projects you create.

    summary

    Create a new project

    Parameters

    • projectPostBodyBody: ProjectPostBodyBody
    • Optional options: AxiosRequestConfig<any>

    Returns Promise<ProjectPostResponse>
